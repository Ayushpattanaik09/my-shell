# Custom Shell in C++

## Objective
To design a mini shell that supports command execution, background jobs, pipes, and redirection.

## Features
- Executes Linux commands
- Supports background processes (&)
- Supports piping (|) and redirection (<, >, >>)
- Displays running jobs
- Built with C++17
